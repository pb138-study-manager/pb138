# Subtasks Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add one-level subtask support — subtasks are full task records linked via `parent_id` FK, created through the existing dialog, displayed inside the parent's card.

**Architecture:** `parent_id` self-referencing FK on `tasks` table. Backend filters top-level tasks in list, cascades deletes, and includes subtasks in detail. Frontend creates subtasks after parent is saved, renders them inline in `TaskCard`.

**Tech Stack:** Drizzle ORM, ElysiaJS, Zod, React, TanStack Query

```typescript
  .get('/:id', async ({ params, user, set }) => {
    const [task] = await db

        and(
          eq(tasks.id, Number(params.id)),

    return { ...task, subtasks };
  })
```

Replace the `DELETE /:id` handler body to cascade:

```typescript
  .delete('/:id', async ({ params, user, set }) => {
    const authUser = user as AuthUser;
    const isAdmin = authUser.roles.includes('ADMIN');
    const [existing] = await db
      .select()
      .from(tasks)
      .where(
        and(
          eq(tasks.id, Number(params.id)),
          isAdmin ? undefined : eq(tasks.userId, authUser.id),
          isNull(tasks.deletedAt)
        )
      );
    if (!existing) {
      set.status = 404;
      return { error: 'NOT_FOUND', message: 'Task not found or access denied' };
    }
    const subtasks = await db
      .select()
      .from(tasks)
      .where(and(eq(tasks.parentId, existing.id), isNull(tasks.deletedAt)));
    for (const sub of subtasks) {
      await db.update(tasks).set({ deletedAt: new Date() }).where(eq(tasks.id, sub.id));
      await logAction(db, authUser.id, `Deleted subtask ${sub.id} (cascade from ${existing.id})`);
    }
    await db.update(tasks).set({ deletedAt: new Date() }).where(eq(tasks.id, existing.id));
    await logAction(db, authUser.id, `Deleted task ${existing.id}`);
    return { success: true };
  })
```

- [ ] **Step 3: Update `PATCH /:id` handler**

Replace the PATCH handler body to handle `parentId`:

```typescript
  .patch(
    '/:id',
    async ({ params, body, user, set }) => {
      const authUser = user as AuthUser;
      const [existing] = await db
        .select()
        .from(tasks)
        .where(
          and(
            eq(tasks.id, Number(params.id)),
            eq(tasks.userId, authUser.id),
            isNull(tasks.deletedAt)
          )
        );
      if (!existing) {
        set.status = 404;
        return { error: 'NOT_FOUND', message: 'Task not found or access denied' };
      }

      if (body.parentId !== undefined && body.parentId !== null) {
        const [parent] = await db
          .select()
          .from(tasks)
          .where(
            and(
              eq(tasks.id, body.parentId),
              eq(tasks.userId, authUser.id),
              isNull(tasks.deletedAt),
              isNull(tasks.parentId)
            )
          );
        if (!parent) {
          set.status = 404;
          return { error: 'NOT_FOUND', message: 'Parent task not found or is itself a subtask' };
        }
      }

      const [updated] = await db
        .update(tasks)
        .set({
          ...(body.title !== undefined && { title: body.title }),
          ...(body.description !== undefined && { description: body.description }),
          ...(body.dueDate !== undefined && { dueDate: new Date(body.dueDate) }),
          ...(body.status !== undefined && { status: body.status }),
          ...('parentId' in body && { parentId: body.parentId }),
        })
        .where(eq(tasks.id, existing.id))
        .returning();
      await logAction(db, authUser.id, `Updated task ${existing.id}`);
      return updated;
    },
    zodBody(UpdateTaskSchema)
  )
```
