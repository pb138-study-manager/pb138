Vyber prístup ktorý ti sedí, alebo kombinuj

A

Solid First

Dni 1–2: Všetky bugy, Courses → API, AuthGuard, vizuálny lesk na kľúčových stránkach

Dni 3–4: Základný AI Copilot panel (daily brief) + quiz z poznámok

Pros

  • Demo bez padania
  • Architektúra obhájiteľná

Cons

  • AI features môžu byť plytké
  • Menej "wow"
B

AI First

Dni 1–2: AI Copilot panel (daily brief + chat) + quiz/chat s poznámkami

Dni 3–4: Bug fixes, Courses → API, polish kľúčových stránok

Pros

  • Wow feature je hlboká a polished
  • Diferenciátor je silný

Cons

  • Riziko nestability v demo
  • Courses stále mock data
C

Paralelné trackery ⭐ Odporúčam

Deň 1: Bug fixes + AuthGuard + Courses → API (1–2 ľudia)

Deň 2: AI backend routes + OpenAI integrácia (1 človek) + visual polish today/tasks/notes (1 človek)

Deň 3: AI Copilot panel frontend + quiz UI v Notes

Deň 4: Chat s poznámkami + full demo run + posledný polish

Pros

  • Využíva 4-členný tím
  • Solid základ + deep AI features
  • Demo flow kompletný

Cons

  • Vyžaduje koordináciu
  • Merge konflikty
