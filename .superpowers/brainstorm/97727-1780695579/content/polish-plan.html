Vizuálny polish — čo a kde

Sekcia 4 — konkrétne vylepšenia na každej stránke demo flow

Demo flow: Login → Today → Tasks → Notes → Timeline → Courses

/login + /register
Stav: ✅ funguje
Čo pridať
  • Gradient hero background (nie flat šedá)
  • Logo / brand name prominent
  • Loading spinner na submit button
Prečo to robí rozdiel

Prvý dojem. Ak login vyzerá genericky, celý produkt pôsobí genericky.

/today
Stav: ✅ API OK
Čo pridať
  • Greeting header: "Dobré ráno, Peter 👋"
  • Progress bar "X z Y úloh hotových"
  • Empty state s CTA ak nie sú úlohy
  • Animated task check (✓ pri toggling)
Prečo to robí rozdiel

Today je prvá stránka po logine — musí okamžite ukázať hodnotu a pocit "toto je moje".

/tasks
Stav: ✅ API OK
Čo pridať
  • Farebné priority badges (červená/žltá/zelená)
  • Deadline countdown "za 2 dni"
  • Skupinové sekcie: Dnes / Tento týždeň / Neskôr
  • Strikethrough animácia pri dokončení
Prečo to robí rozdiel

Farebné priority a countdown pridávajú urgentnosť — tasks stránka sa cíti živá, nie len zoznam.

/notes
Stav: ✅ API OK
Čo pridať
  • 🧠 Quiz me + ✦ Ask AI tlačidlá (AI features)
  • Word count + čas čítania v headerri
  • Folder farby (nie len ikony)
Prečo to robí rozdiel

Notes je kde NotebookLM feature žije — musí pôsobiť ako skutočný editor, nie textové pole.

/timeline
Stav: ⚠️ čiastočne
Čo pridať
  • Farebné kategórie eventov (kurz / osobné / deadline)
  • Hover tooltip s detailom eventu
  • Tlačidlo "Pridať event" prominentné
Prečo to robí rozdiel

Timeline je vizuálne najimpresívnejšia stránka ak je nabitá dátami — perfektná pre demo.

/courses
Stav: ❌ mock dáta
Čo treba
  • Pripojiť na reálne API (useCourseManager hook)
  • Progress bar per kurz (% úloh hotových)
  • Farebné kurz karty (každý kurz = iná farba)
  • Enroll / unenroll button funkčný
Prečo to robí rozdiel

Ak kurzy fungujú len s mock dátami, pedagóg si to všimne pri otázkach. Reálne API je nevyhnutné.

Bonus — seed dáta pre demo: Vytvoriť seed script ktorý naplní DB realistickými dátami (3 kurzy, 10 úloh, 5 eventov, 4 poznámky). Demo nikdy nebehá na prázdnej DB.
